HTML Tags

    HTML tags are the keywords that are used to produce web pages in various formats. Opening tags and closing tags are found in the majority of tags. The concluding tags contain a forward slash (/), while the language of the beginning tags is the same. Certain tags don't need to be closed.

  1. Html: The html tag in HTML is used to define the root of HTML and XHTML documents.
  2. Head: The head tag in HTML is used to define the head portion of the document which contains information related to the document.
  3. Body: The body tag in HTML is used to define the main content present inside an HTML page.
  4. Bold: The bold tag in HTML is used to specify the bold text without any extra importance.
  5. Break: The break tag inserts a single carriage return or breaks in the document. This element has no end tag.
  6. Comment: The comment tag is used to insert comments in the HTML code.
  7. Div: The div tag is used in HTML to make divisions of content in the web page (text, images, header, footer, navigation bar, etc)
  8. Hr: The hr tag in HTML stands for horizontal rule and is used to insert a horizontal rule.
